### **Setting Up Alerts**:
1. Click bell icon in top toolbar
2. Select alert type (Price, RSI, MACD, etc.)
3. Set threshold/condition
4. Choose delivery method (Browser, Email, SMS)
5. Save alert
6. View active alerts in right panel

### **Using Watchlist**:
1. Click "+ Add" in watchlist panel
2. Enter symbol (e.g., TSLA)
3. Symbol appears in list with live quote
4. Click symbol to load chart
5. Drag to reorder
6. Click X to remove

### **Comparing Symbols**:
1. Click "Compare" button in chart controls
2. Enter symbols to compare (e.g., AAPL, MSFT, GOOGL)
3. Choose view: Overlay or Normalized %
4. All symbols plotted on same chart
