## How to Use

### Add Any ETF

1. **Via Search Bar**: Type symbol (e.g., "SCHD") and click "Add ETF"
2. **Via Quick-Add**: Click any categorized button
3. **Manual Entry**: Type symbol and press Enter

## API Endpoints

All ETFs accessible via:
- `GET /api/etf-analyzer/profile/:symbol` - ETF profile with live price
- `GET /api/etf-analyzer/holdings/:symbol` - Holdings (if available)
- `POST /api/etf-analyzer/overlap` - Compare holdings overlap
- `POST /api/etf-analyzer/compare-expenses` - Compare expense ratios
